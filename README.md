eICU sepsis ED-to-ICU delay analysis
This repository contains the analysis code for:
Assessing ICU admission delays in sepsis: a generalized additive model analysis of mortality outcomes
The analysis uses the eICU Collaborative Research Database and evaluates the association between emergency department (ED) length of stay before ICU admission and ICU mortality among ICU patients with sepsis or related severe infection syndromes.
Data access
The raw eICU Collaborative Research Database is not included in this repository.
Access to eICU requires PhysioNet credentialing. After obtaining access, place the eICU-CRD 2.0 files locally and point the scripts to that folder using the `EICU_DIR` environment variable.
Expected raw files include:
`patient.csv.gz`
`treatment.csv.gz`
`apachePatientResult.csv.gz`
`pastHistory.csv.gz`

How to run
From the repository root:
```r
Sys.setenv(EICU_DIR = "/path/to/eicu-crd/2.0")
source("scripts/run_all.R")
```
Or run scripts one by one:
```r
source("scripts/01_clean_data.R")
source("scripts/02_table1.R")
source("scripts/03_models.R")
source("scripts/04_tables_export.R")
```
Scripts
Script	Purpose
`01_clean_data.R`	Builds the analytic cohort; derives ED LOS, vasopressor exposure, APACHE IV, Charlson score, and outcome; exports flowchart numbers and audit outputs, including the Charlson/pastHistory audit.
`02_table1.R`	Generates baseline characteristics stratified by ICU mortality.
`03_models.R`	Fits logistic regression and GAM models, performs validation, DeLong tests, interaction tests, AIC comparisons, and exports figures.
`04_tables_export.R`	Formats final manuscript and supplementary tables.
`run_all.R`	Runs the main pipeline in order.

Reproducibility notes
ED LOS is calculated as `-hospitaladmitoffset / 60`, without using absolute values.
ED LOS is restricted to values between 0 and 24 hours.
The Charlson score is not age-adjusted in the main analysis because age is included as a separate model covariate.
Internal validation uses a stratified 70/30 split by ICU mortality status with fixed random seed.
Logistic regression and GAM models are compared with and without comparable ED LOS × vasopressor interaction structures.

License and data restrictions
Only analysis code should be shared in this repository. Do not upload raw eICU data or patient-level derived datasets to GitHub.
Charlson audit
The Charlson/pastHistory audit is now generated directly by `scripts/01_clean_data.R` when `RUN_CHARLSON_AUDIT <- TRUE`. This keeps cohort construction and Charlson diagnostics in one reproducible entry-point script.
Figures
The analysis pipeline generates the two model-based manuscript figures: `Figure2_gam_main_v2.png` and `Figure3_gam_interaction_v2.png`. No ROC figure or supplementary figure is generated by default, because model discrimination is reported in the validation performance and DeLong comparison tables.
