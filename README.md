This repository contains the reproducible R analysis code for the manuscript:

> **Assessing ICU admission delays in sepsis: a generalized additive model analysis of mortality outcomes**

The study evaluates whether emergency department (**ED**) length of stay before ICU admission is associated with ICU mortality among patients with sepsis or related severe infection syndromes. It also compares conventional logistic regression and generalized additive models (**GAMs**) with and without an ED LOS × vasopressor interaction.

---

## Overview

This analysis uses the **eICU Collaborative Research Database** to build a multicenter cohort of ICU admissions originating from the ED. The main outcome is **ICU mortality**. The main exposure is **ED length of stay before ICU admission**, restricted to values between 0 and 24 hours.

The analytical workflow includes:

1. Cohort construction from raw eICU tables.
2. Derivation of ED length of stay, APACHE IV score, vasopressor exposure, Charlson Comorbidity Index, and ICU mortality.
3. Baseline cohort description.
4. Multivariable logistic regression and GAM modeling.
5. Internal validation using a stratified 70/30 split.
6. Export of manuscript tables, supplementary tables, and model-based figures.

## Data access

The raw **eICU Collaborative Research Database** is **not included** in this repository.

Access to eICU requires credentialing through PhysioNet. After obtaining access, place the eICU-CRD 2.0 files locally and point the pipeline to that folder using the `EICU_DIR` environment variable.

The analysis expects the following raw eICU files:

```text
patient.csv.gz
treatment.csv.gz
apachePatientResult.csv.gz
pastHistory.csv.gz
```

A convenient local layout is:

```text
data/raw/eicu-crd-2.0/
├── patient.csv.gz
├── treatment.csv.gz
├── apachePatientResult.csv.gz
└── pastHistory.csv.gz
```

## Quick start

From the repository root:

```bash
export EICU_DIR="/path/to/eicu-crd/2.0"
Rscript scripts/run_all.R
```

For example:

```bash
cd /home/m.lopez/eicu_sepsis_analysis
export EICU_DIR="/home/m.lopez/eicu_sepsis_analysis/data/raw/eicu-crd-2.0"
Rscript scripts/run_all.R
```

Alternatively, the scripts can be run one by one in R:

```r
source("scripts/01_clean_data.R")
source("scripts/02_table1.R")
source("scripts/03_models.R")
source("scripts/04_tables_export.R")
```

## Analysis pipeline

| Step | Script | Main purpose |
|---:|---|---|
| 1 | `01_clean_data.R` | Builds the analytic cohort, calculates ED LOS, derives vasopressor exposure, APACHE IV, Charlson score, ICU mortality, flowchart counts, and audit files. |
| 2 | `02_table1.R` | Generates baseline characteristics stratified by ICU mortality. |
| 3 | `03_models.R` | Fits logistic regression and GAM models with and without ED LOS × vasopressor interaction; performs validation and model comparison; exports Figures 2 and 3. |
| 4 | `04_tables_export.R` | Formats final manuscript and supplementary tables. |
| 5 | `run_all.R` | Runs the full pipeline in order. |

## Reproducibility notes

Important implementation choices are fixed in the scripts:

- ED LOS is calculated as `-hospitaladmitoffset / 60`.
- Absolute values are **not** used when calculating ED LOS.
- ED LOS is restricted to values between 0 and 24 hours.
- The main Charlson score is **not age-adjusted**, because age is included separately in the multivariable models.
- Internal validation uses a stratified 70/30 split by ICU mortality status.
- Logistic regression and GAMs are compared using comparable main-effect and interaction structures.
- Candidate models are evaluated using AUROC, Brier score, calibration intercept, calibration slope, DeLong tests, interaction tests, and AIC.

---

## Model specifications

The candidate models are:

1. Logistic regression, main effects.
2. Logistic regression with ED LOS × vasopressor interaction.
3. GAM with ED LOS as a smooth term.
4. GAM with separate ED LOS smooths by vasopressor status.

All candidate models adjust for:

```text
age
sex
ethnicity
APACHE IV score
Charlson Comorbidity Index
vasopressor use within the first 24 hours of ICU admission
ED length of stay before ICU admission
```
